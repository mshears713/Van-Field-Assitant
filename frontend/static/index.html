Audio Upload

Not Built

Audio upload is not implemented yet

This feature requires HTTPS for browser microphone access and a local Whisper installation for transcription. Both are planned for a future version.

Planned audio flow

  • Record voice note on Android phone (using the phone's own voice recorder)
  • Upload the audio file here to the dashboard
  • Backend saves raw audio to notes/inbox/
  • Local Whisper transcribes the audio (CPU-only, small model)
  • Raw transcript is preserved alongside original audio
  • Capture Agent classifies the transcript into a structured note
  • User reviews and promotes note to notes/ready_for_notion/

Current workaround

Use the Agents tab → Capture Agent. Paste your raw voice transcript or notes manually and the Capture Agent will classify and structure them for you.

Network switching is manual

Access from Android phone

  • Android phone and mini PC must be on the same Wi-Fi network
  • Use the Android Access URL shown above in your phone's browser
  • If it does not load, check Windows Firewall (see below)

Windows Firewall

Windows Firewall may block port 8080. Run this command in a terminal as administrator:

netsh advfirewall firewall add rule name="FieldAssistant" dir=in action=allow protocol=TCP localport=8080

Network modes (manual)

Local Field Mode: Mini PC creates its own Wi-Fi (e.g., FieldAssistantNet). Android connects to it. No internet access.

Internet Sync Mode: Mini PC connects to iPhone hotspot temporarily to pull repos or models. The Android dashboard will disconnect during this switch. Reconnect after sync is done.
